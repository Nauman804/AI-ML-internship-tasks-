# 🚀 AI/ML Engineering Internship Portfolio - DevelopersHub Corporation

Welcome to my official internship repository! This project showcases a series of technical milestones completed during my **AI/ML Engineering Internship** at **DevelopersHub Corporation**. These tasks demonstrate my expertise in Data Science, Machine Learning deployment with Flask, and Generative AI integration.

---

## 🛠️ Project Modules & Technical Breakdown

### 📍 Task 1: Advanced Visual Analytics
**Objective:** Perform deep exploratory data analysis and visualization using a custom-curated dataset.
* **Key Achievements:**
    * Generated a **custom dataset** to simulate real-world data constraints.
    * Developed comprehensive visualizations including Correlation Heatmaps, Distribution Plots, and Box Plots.
    * Automated data profiling using `Pandas` and `Seaborn`.

### 📍 Task 3: Heart Disease Prediction (Full Stack ML)
**Objective:** End-to-end Machine Learning pipeline—from model training to a functional Web Interface.
* **Model Training:** Built a classification model to predict heart disease risk based on clinical parameters.
* **Deployment (Flask):** Created a **Flask-based web application** to serve the model.
* **Frontend:** Developed a user-friendly interface using **HTML & CSS** for users to input data and get real-time predictions.
* **Skills:** Model Serialization (Pickle/Joblib), Flask Routing, and Feature Importance analysis.

### 📍 Task 4: AI Health Chatbot (GenAI Web App)
**Objective:** A high-performance, real-time Health Assistant powered by LLMs and modern Web technologies.
* **Inference Engine:** Integrated **Groq API** (Llama-3/Mixtral) for instant response generation.
* **Full-Stack Implementation:** * **Backend:** Python logic to handle API requests.
    * **Frontend:** Interactive UI built with **HTML, CSS, and JavaScript**.
    * **Notebooks:** `.ipynb` files documenting the prompt engineering and API testing phase.
* **Prompt Engineering:** Optimized system prompts to ensure the bot acts as a professional and safe medical assistant.
